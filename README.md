# degen
react native expo degen app

https://docs.expo.dev/get-started/create-a-new-app/
Follow expo setup instructions to get expo cli installed on your machine. Install Expo Go from app store on your device if wanting to run on real device.

yarn && yarn start to start metro bundler
Best to run on ios simulator or real ios device as Android has an issue with Intl library on dev version.
